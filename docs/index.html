 ログイン

匿名の影に隠れない、責任あるコミュニティを。

MIRAIは、Discordサーバーにおける認証BOTです。

オンライン上の交流は自由である一方、過度な匿名性が無責任な言動や不正行為を生み出す温床にもなり得ます。

MIRAIは、ユーザーに認証を求めることで匿名性の悪用を排除し、サーバー管理者と利用者が共に安心できる環境を提供します。

Features

ウェブダッシュボードによる直感的な管理

MIRAIは専用のウェブダッシュボードを提供しており、サーバー管理者はすべての認証設定をブラウザ上から簡単に操作できます。

複雑なコマンド操作は不要。GUIベースで効率的に管理が可能です。

不正アクセス・自動認証防止

VPN・プロキシ検知機能

匿名化サービスを経由したアクセスを自動的に拒否し、不正利用を未然に防ぎます。

ボット検知機能

ロボットによる不正な認証リクエストを遮断し、人間の利用者のみを通過させます。

強固なセキュリティ

二要素認証（2FA）対応

ダッシュボードへのアクセスには二要素認証を必須化可能。

管理者アカウントの乗っ取りや不正ログインを強力に防ぎます。

コミュニティの特性に応じた制御

国・言語フィルタ

国籍や使用言語を指定し、国内コミュニティに対する海外からのアクセスを制限できます。

サーバーの目的に応じた利用者層を維持可能です。

年齢制限チェック

Discordが定める対象年齢に満たないユーザーを自動的にブロック。

規約順守と健全なコミュニティ運営をサポートします。

MIRAIが実現するもの

安心・安全な交流環境

認証を通じて、サーバー内のユーザーが実在する人物であることを確認できます。これにより、荒らし行為やスパムのリスクを大幅に軽減し、安心して交流できる環境を提供します。

コミュニティの質の向上

認証プロセスを導入することで、真剣に参加したいユーザーのみがアクセスできるようになります。これにより、コミュニティの質が向上し、有意義な交流が促進されます。

管理者の負担軽減

MIRAIは自動化された認証システムを提供するため、サーバー管理者の手動によるユーザー管理の負担を軽減します。これにより、管理者はコミュニティ運営に集中できるようになります。
