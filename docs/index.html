人間であるかの検証や、VPNの検出、危険なアカウントの制限など、セキュリティに関する機能を提供します。

Features

データフォーム

必要なデータを入力して認証を行います。

人間検証

人間であるかどうかを検証します。

VPN検出

VPNを使用しているかどうかを検出します。

危険なアカウントの制限

ハッキングされたアカウントや、悪意のあるアカウントを制限します。

デバイス認証

デバイスの識別情報を取得して認証を行います。

行動パターン分析

ユーザーの行動パターンを分析して異常を検出します。

二要素認証

別の方法も加えて認証を行います。
