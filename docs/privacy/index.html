プライバシーポリシー

運営者は、本サービスを運営するうえで、利用者の個人情報を取得することがあります。

テキストチャンネルでのチャットデータはコマンドの実行にのみ使用します。

ボイスチャンネルでの音声データを取得することはありません。

取得した個人情報は、本サービスの運営にのみ使用し、第三者に提供することはありません。

第1条（個人情報の定義）

本ポリシーにおいて「個人情報」とは、個人情報保護法第2条第1項に定義される「個人情報」を指します。

第2条（個人情報の取得）

運営者は、利用者が本サービスを利用する際に、以下の情報を取得します。

  1. DiscordのユーザーID
  2. Discordのユーザー名
  3. Discordの登録に使用したメールアドレス
  4. ユーザーに紐づけられたIPアドレスとその取得日時
  5. ユーザーに紐づけられた誕生日

運営者は、利用者が本サービスを利用する際に、上記の情報を取得します。

第3条（個人情報の利用目的）

運営者は、利用者の個人情報を以下の目的で利用します。

  1. 本サービスの提供・運営
  2. 利用者の本人確認
  3. 利用者の年齢確認
  4. 利用者からのお問い合わせに対する回答
  5. 本サービスに関する情報の提供
  6. 本サービスの改善・新サービスの開発
  7. 利用者の利用状況の分析
  8. 犯罪行為が判明した際に、裁判所や警察等の公的機関に対して迅速に情報を提供するため
  9. その他、運営者が必要と判断した場合

第4条（個人情報の管理）

運営者は、利用者の個人情報を適切に管理し、漏洩、改ざん、紛失等の防止に努めます。

また、運営者は、利用者の個人情報を適切に管理するために、以下の措置を講じます。

  1. 個人情報の取り扱いに関する教育・研修の実施

第5条（個人情報の第三者提供）

運営者は、利用者の個人情報を第三者に提供することはありません。

ただし、以下の場合は除きます。

  1. 利用者の同意がある場合
  2. 法令に基づく場合
  3. 人の生命、身体または財産の保護のために必要な場合
  4. 公衆衛生の向上または児童の健全な育成に寄与するために特に必要な場合
  5. 国の機関または地方公共団体等が法令の定める事務を遂行するために協力する必要がある場合
  6. その他、運営者が必要と判断した場合

第6条（個人情報の開示）

運営者は、利用者から個人情報の開示を求められた場合、以下の手続きに従って開示します。

  1. 開示を求める個人情報の特定
  2. 開示を求める利用者の本人確認(身分証明書の提示が必要)
  3. 求められた個人情報の開示

ただし、以下の場合は開示を行わないことがあります。

  1. 利用者本人であることが確認できない場合
  2. 開示を求める個人情報が存在しない場合
  3. 開示を行うことで、他の利用者または第三者の権利を侵害する恐れがある場合
  4. 法令に違反する恐れがある場合
  5. その他、運営者が必要と判断した場合

第7条（個人情報の訂正・削除）

運営者は、利用者から個人情報の訂正または削除を求められた場合、以下の手続きに従って対応します。

  1. 訂正または削除を求める個人情報の特定
  2. 訂正または削除を求める利用者の本人確認(身分証明書の提示が必要)
  3. 求められた個人情報の訂正または削除

ただし、以下の場合は訂正または削除を行わないことがあります。

  1. 利用者本人であることが確認できない場合
  2. 訂正または削除を求める個人情報が存在しない場合
  3. 訂正または削除を行うことで、他の利用者または第三者の権利を侵害する恐れがある場合
  4. 法令に違反する恐れがある場合
  5. その他、運営者が必要と判断した場合

第8条（個人情報の保護に関する法令等の遵守）

運営者は、個人情報の保護に関する法令及びその他の規範を遵守します。

第9条（プライバシーポリシーの変更）

運営者は、必要に応じて本ポリシーを変更することがあります。

第10条（お問い合わせ）

本ポリシーに関するお問い合わせは、以下のメールアドレスまでお願いいたします。

info@mirai.jun-suzu.net

第11条（準拠法）

本ポリシーは、日本法に準拠します。

第12条（管轄裁判所）

本ポリシーに関する紛争については、運営者の本社所在地を管轄する裁判所を専属的合意管轄裁判所とします。
