プライバシーポリシー

最終更新日: 2025年5月5日

本プライバシーポリシーは、Discord認証BOT「MIRAI」（以下、「当BOT」）をご利用いただく際に、ユーザーの皆様から取得する情報の種類、利用目的、およびその取り扱い方について説明するものです。

1. 取得する情報

当BOTでは、以下の情報を取得します：

  • Discordユーザー名およびDiscord ID
  • Discordに登録されたメールアドレス
  • 認証時にユーザーが入力する誕生日
  • 接続に使用されたIPアドレス

2. 情報の利用目的

取得した情報は、以下の目的で利用されます：

  • ユーザー名およびID：認証およびユーザー識別のため
  • メールアドレス：トラブル対応時に直接ご連絡するため（DMが無効な場合に備えて）
      ※「info[at]jun-suzu.net」からのメールを受信できるよう設定をお願いします。
  • 誕生日：Discordの利用規約に定められた「13歳以上」の条件を満たしているかを確認するため
      ※認証時に13歳未満であっても、13歳に達すると自動で認証が通ります。
  • IPアドレス：
    • VPN利用の判定およびアクセス元国の特定のため（VPN経由の認証は原則として不可）
    • 荒らし行為やポルノ投稿など、悪質な行為が繰り返され、かつDiscord運営による対応が不十分と判断された場合、該当IPアドレスを通報先ISPに提供することがあります。被害を受けたサーバーの管理者様は、MIRAI荒らし対応ガイドラインを参照の上、通報を行ってください。

3. 情報の保管と管理

取得した情報は、適切なセキュリティ措置のもとで管理され、不正アクセス、改ざん、漏洩等のリスクを最小限に抑えるよう努めています。

4. 第三者提供

原則として、ユーザーの同意なく第三者に個人情報を提供することはありません。ただし、以下の場合は例外とします：

  • 法令に基づく場合
  • 前述のような悪質行為により、対応を必要とする場合

5. AI学習へのデータ利用について

当BOTでは、サービスの品質向上およびAI技術の改善を目的として、ユーザーとのテキストチャット内容を匿名化した上で分析または機械学習に活用する場合があります。

利用にあたっては、個人を特定できる情報（氏名、メールアドレス、Discord IDなど）は含めないか、加工・除去した上で使用します。

こうした利用を希望されない場合は、info[at]jun-suzu.netまでご連絡いただければ、対象から除外いたします。

ただし、すでに学習済みのデータについては、除外が難しい場合がありますので、ご了承ください。

6. 情報の修正・削除

取得した情報の訂正や削除をご希望の場合、本人確認のために公的な身分証明書の提出をお願いすることがあります。これは、本人なりすましによる不正な情報改ざんを防止するための措置です。

  • 本人確認に必要な情報は、本人確認が完了次第、速やかに削除します。
  • 提出された情報は、本人確認以外の目的では絶対に使用しません。
  • 本人確認に使用できる身分証明書の例としては、運転免許証、パスポート、マイナンバーカードなどがあります。

7. お問い合わせ

本ポリシーに関するご質問や情報取り扱いについてのお問い合わせは、以下のメールアドレスまでご連絡ください：

📧 info[at]jun-suzu.net ※スパム対策のため、＠を[at]に置き換えています。
