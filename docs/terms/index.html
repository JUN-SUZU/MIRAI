MIRAI BOT 利用規約約款

最終更新日: 2024年5月5日

この利用規約（以下「本規約」といいます）は、「JUN-SUZU」が開発・提供するDiscord認証BOT「MIRAI」（以下「当BOT」といいます）の利用に関する条件を定めるものです。当BOTを利用する前に必ずお読みください。

第1条（利用目的）

当BOTは、Discordサーバーにおいてユーザーの年齢確認およびVPNを除外した認証を行うことで、荒らし行為や違法行為の防止を目的としています。完全な匿名性を排除し、「責任ある利用環境」の維持を目指します。

第2条（利用条件）

  • 当BOTは13歳以上の方のみが利用できます。保護者の同意の有無にかかわらず、13歳未満の方はご利用いただけません。
  • 入力された年齢情報により、13歳に達したと判定された時点で認証が可能となります。
  • 当BOTは、VPNを利用しているユーザーの認証を原則として行いません。VPNを利用している場合は、VPNを無効にしてから再度認証を行ってください。

第3条（禁止事項）

当BOTの利用にあたり、以下の行為を禁止します：

  1. 虚偽の情報提供（偽の年齢入力、偽の通報等）
  2. BOTのサーバーやシステムへの不正アクセス
  3. 当BOTに搭載されたAIの解析や回避的使用（スパム検知回避を目的とした試行）
  4. 当BOTが導入されているサーバー内での誹謗中傷・迷惑行為
  5. その他、公序良俗または法令に反する行為

第4条（利用制限）

ユーザーが本規約に違反した場合、または不正利用が認められた場合、予告なく当BOTの利用を制限または停止することがあります。再認証や解除を希望する場合は、下記の連絡先にご相談ください。

第5条（免責事項）

  • 当BOTは常に安定した動作を保証するものではありません。
  • BOTの不具合、停止、またはそれによって生じた損害について、開発者は一切の責任を負いません。
