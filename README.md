# MIRAI
## Be more secure with MIRAI
### What is MIRAI?
MIRAI is a simple tool to help you manage your discord server. It is a bot that can do many things, such as:
- Moderation
- Authentication
- Detection of spam with high level AI
- And more!

### How to use MIRAI?
All you need to do is to invite MIRAI to your server and type `/setup` to get started!
It will take you to a good future of your server!
